Reflection

In coding this portfolio, two web development topics I adored using were flexbox and pseudo-classes.

Flexbox

The flexible box, or flexbox layout, organizes HTML sections within a container. It is used for efficient spacing of items, also helping with responsive browser sizes.

With cohesive properties such as arranging item order, direction, and spacing, flexbox assists with making a site structured according to the developer's liking.

I like it as it is simply the best and simplest in organizing a site! The syntax is simple to remember and not as convoluted as CSS grid. I also like how it makes items responsive.

Learn more: A Complete Guide to Flexbox

Pseudo-classes

A pseudo-class allows selectors to be added to elements. Pseudo-classes are fun as they create easter eggs for a site, or increase visual stimulation. Popular ones include :hover and :active, as they can change the style of text, links, photos and more media.

I like it because it adds interactivity to a site! At a stage where I haven't learned Javascript, pseudo-classes make my projects stimulating and more creative.

Learn more: MDN Pseudo-classes
